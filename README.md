## Описание Проекта
__Используемые технологии:__ JS, NodeJS, express, GIT.

## Локальный запуск
1. Склонировать репозиторий
2. Доставить отсутствющие модули npm
    ```
        npm install
    ```
3. Запустить локальный сервер
    ```
        npm run dev
    ```
